How to Access and Use This Code
===============================

This code is available freely on GitHub under the MIT license. The data used to
train the models is available upon request to L. Pereira Diaz.

From the command line, the code can be downloaded using ``git``:

.. code:: bash

    $ git clone https://github.com/bionictoucan/multi_pow .

This will create a directory within your current directory called ``multi_pow``
which includes all the code documented here.
